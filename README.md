 📈 Stock Price Analysis Using Python

## Project Overview

This project analyzes historical stock market data using Python to uncover trends, patterns, and market behavior. The analysis includes data preprocessing, feature engineering, time-series analysis, and data visualization.

Objectives

- Clean and preprocess stock market data
- Perform data transformation using Pandas
- Conduct time-series analysis
- Create meaningful financial features
- Visualize stock performance
- Generate actionable insights

 Dataset

The dataset contains the following variables:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Trading Volume

 Tools & Libraries

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

Data Transformation

The following preprocessing steps were completed:

- Converted Date column to datetime format
- Checked for missing values
- Sorted data by date
- Created Month and Year columns
- Grouped and aggregated monthly statistics
- Prepared the dataset for time-series analysis

Feature Engineering

The following features were created:

- Daily Price Change
- Percentage Price Change
- 7-Day Moving Average (MA7)
- 30-Day Moving Average (MA30)
- Daily Return
- Monthly Return
- Rolling Volatility
- High-Low Price Difference

Visualizations

The project includes:

- 📈 Stock Closing Price Trend
- 📊 Trading Volume Trend
- 📉 Moving Average Analysis
- 📊 Daily/Monthly Return Analysis

 Key Insights

- Stock prices experienced both upward and downward trends over the observed period.
- Trading volume fluctuated, reflecting changes in market participation.
- Short-term moving averages reacted faster to market movements than long-term averages.
- Periods of higher volatility coincided with larger price swings.



Recommendations

- Monitor both short-term and long-term moving averages before making investment decisions.
- Consider trading volume together with price movement for better market interpretation.
- Be cautious during periods of high volatility.
- Continue tracking monthly returns to identify recurring trends.
